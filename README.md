Initialize project venv and install/update dependencies:

    dev/setup

Run... something:

    dev/run

Test/lint:

    dev/test
    dev/lint

Install/uninstall a package in the dev environment and update requirements.txt:

    dev/install requests
    dev/uninstall requests
